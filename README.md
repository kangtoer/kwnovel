# ✨ Features

- 🎨 Tampilan Visual Novel Retro 32-Bit
- 🏰 Latar sejarah Nusantara yang dinamis
- 🎵 Sound Effect menggunakan Tone.js
- 💬 Dialog Typewriter Animation
- ❤️ Sistem HP / Energi Temporal
- 🏺 Sistem Koleksi Artefak
- 📚 Materi Sejarah Indonesia
- ❓ Soal HOTS Interaktif
- 🌏 Responsive Desktop & Mobile
- ⚡ Tanpa Framework JavaScript
- 🎯 Single HTML File

---

# 📖 Story

Sebagai seorang **Penjelajah Waktu**, pemain ditugaskan mengumpulkan artefak sejarah Nusantara.

Namun...

Setiap zaman memiliki penjaga sejarah.

Untuk memperoleh artefak, pemain harus mampu menjawab berbagai pertanyaan sejarah yang diajukan tokoh-tokoh penting Indonesia.

Kesalahan akan mengurangi energi temporal.

Jika energi habis...

Pemain akan terjebak di masa lalu selamanya.
